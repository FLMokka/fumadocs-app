## Usage & Examples

1. ### With Title (Collapsed by Default, Collapsible)
Creates a collapsible code block that starts in the collapsed state, shows a header with title and collapse functionality.

```tsx
import { CodeBlock } from "@/components/ui/code-block";

<CodeBlock
  code={exampleCode}
  title="useGoogleAuth.ts"
  colorScheme={theme === "dark" ? "dark" : "light"}
/>
```

<CodeBlockDemo caseNumber={1} />

2. ### With Title (Expanded by Default, Collapsible)
When no title is provided, creates a collapsible code block that starts expanded setting <code>defaultOpen</code> to <code>true</code>, shows a header with title and collapse functionality.

```tsx
<CodeBlock
  code={exampleCode}
  title="Supabase OAuth Sign-in"
  defaultOpen
/>
```

<CodeBlockDemo caseNumber={2} />

3. ### Without Title (Always Expanded)
Creates a simple code block that is always visible, no collapse functionality.

```tsx
<CodeBlock
  code={exampleCode}
/>
```
